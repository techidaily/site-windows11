## How to Fix the ‚ÄúSemaphore Timeout Period Has Expired‚Äù Error 0X80070079 in Windows 11/10

The error 0x80070079 message says, ‚ÄúThe semaphore timeout period has expired.‚Äù As a result, users can‚Äôt copy or move files between their PCs and storage devices.

## 1\. Try Transferring or Copying Files in Smaller Groups

 It might be the case that the 0x80070079 error occurs on your PC because of an issue with one particularly large file in a group. So, try transferring (or copying) a smaller group of files to see if that makes a difference. If that works, move or copy more files of the files you need within another smaller batch.

 Error 0x80070079 can arise if there‚Äôs an issue with the cable you‚Äôre using for connecting an external USB storage drive to the computer. Try utilizing a different USB cable if you have one available. If you can see your current cable is visibly damaged, consider purchasing a replacement.
