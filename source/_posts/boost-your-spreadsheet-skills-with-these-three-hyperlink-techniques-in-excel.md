 For quick access to a cell, document, or webpage, you can insert hyperlinks in your Excel spreadsheet. Just paste in a hyperlink, give it a readable label, and click to follow the link. Here are a few ways to insert hyperlinks in Excel.

 You can use [Excel's built-in link feature](https://tech-revival.techidaily.com/enhance-your-at-home-workflow-with-these-6-ingenious-uses-of-chatgpt/), the HYPERLINK function, or simply enter a [URL](https://extra-information.techidaily.com/boost-your-competitive-edge-with-custom-vocal-alteration-techniques-in-free-fire-gaming-no-charge/) that converts to a link automatically. Let's look at each of these methods so you can use the one that works best for you.

##  Use the Link Feature

 Like other Office applications such as Word and PowerPoint, Excel offers a link tool that gives you some flexibility. You can link to an existing file, web page, [location in a sheet](https://facebook-video-footage.techidaily.com/updated-2024-approved-dynamic-and-simple-building-a-subscriber-button-for-youtube-using-filmora/), or an email address. You can even insert a link to create a new document.

Related: [How to Link to Another Sheet in Microsoft Excel](https://facebook-video-footage.techidaily.com/updated-2024-approved-dynamic-and-simple-building-a-subscriber-button-for-youtube-using-filmora/) 

 Go to the cell where you want to add the link. Open the Insert tab, select the Link drop-down menu, and choose "Insert Link." Alternatively, [right-click](https://desktop-recording.techidaily.com/updated-the-art-of-recording-fun-6-techniques-to-document-minecraft-for-2024/), move to Link, and pick "Insert Link."

![Insert Link in the Excel shortcut menu](https://static1.howtogeekimages.com/wordpress/wp-content/uploads/2022/12/LinkInsert-ExcelWaysInsertLinks.png) 

 When the Insert Hyperlink box opens, you'll see your link options.

**Existing File or Web Page**: Choose a file from the current folder or list of recent files or pick a web page from browsed pages or enter the URL in the Address box.

![Link to an existing file or web page](https://static1.howtogeekimages.com/wordpress/wp-content/uploads/2022/12/ExistingFileWebPage-ExcelWaysInsertLinks.png) 

**Place in This Document**: Select the sheet and enter the cell in the box above or choose a [defined name](https://fox-blue.techidaily.com/2024-approved-mastering-chroma-key-techniques-in-live-broadcasts/).

![Link to a place in this document](https://static1.howtogeekimages.com/wordpress/wp-content/uploads/2022/12/PlaceInDocument-ExcelWaysInsertLinks.png) 

**Create a New Document**: Give the document a name, optionally change the path, and choose when to edit the new document.

![Link to create a new document](https://static1.howtogeekimages.com/wordpress/wp-content/uploads/2022/12/CreateNewDocument-ExcelWaysInsertLinks.png) 

**Email Address**: Select a recently used email address or enter one near the top. Optionally enter a subject line. You'll only see this option if you save the workbook locally rather than [automatically to OneDrive](https://tech-haven.techidaily.com/simplifying-life-comparing-claude-and-chatgpt-skills/).

![Link to an email address](https://static1.howtogeekimages.com/wordpress/wp-content/uploads/2022/12/EmailAddress-ExcelWaysInsertLinks.png) 

**Text to Display**: At the top of the Insert Hyperlink box, you can enter the display text you want to use for the link rather than showing the link itself.

![Link text to display](https://static1.howtogeekimages.com/wordpress/wp-content/uploads/2022/12/TextToDisplay-ExcelWaysInsertLinks.png) 

 Once you complete the fields for the option you pick above, click "OK" to insert the link in the cell.

![Link inserted with the path on hover](https://static1.howtogeekimages.com/wordpress/wp-content/uploads/2022/12/LinkInserted-ExcelWaysInsertLinks.png) 

 You can hover your cursor over the hyperlink to see the path.

##  Insert the HYPERLINK Function

 An alternative to the link feature is the HYPERLINK [function in Excel](https://vimeo-videos.techidaily.com/in-2024-high-end-downloads-best-10-apps-for-extracting-vimeo-videos/). With this function, you can create a custom formula that links to an existing sheet, workbook on your intranet, or file on another drive. You can also link to a Word document or directly to a [Word bookmark](https://win-solutions.techidaily.com/stop-interruptions-ultimate-solution-to-world-of-tanks-blitz-stability-problems/).

 The syntax for the formula is

        `HYPERLINK(location, text)`
    
 where the first argument is required and contains the full path and file name as the link. For the second argument, you can enter text in quotes or a cell reference to display friendly text for the link.

 To link to cell A1 in Sheet2 within the current workbook (NewBook.xlsx) with the display text Click Here, you'd use this formula:

=HYPERLINK("[NewBook.xlsx]Sheet2!A1","Click Here")

![HYPERLINK function for another sheet](https://static1.howtogeekimages.com/wordpress/wp-content/uploads/2022/12/HyperlinkFunctionSheet-ExcelWaysInsertLinks.png) 

 To link to cell A1 in Sheet2 in a workbook on the web (in our case it's located at `https://oursite.com/report.xlsx`) and display the text View Report, you'd use this formula.

=HYPERLINK("[https://oursite.com/report.xlsx]Sheet2!A1","View Report")

![HYPERLINK function for a file on the web](https://static1.howtogeekimages.com/wordpress/wp-content/uploads/2022/12/HyperlinkFunctionWeb-ExcelWaysInsertLinks.png)

##  Type or Paste a URL

 If you want to add a hyperlink in Excel for a website or page, you can simply enter the URL or [paste it](https://facebook-videos.techidaily.com/new-ultimate-collection-20-chuckle-inducing-detention-cell-photos-for-a-better-day-for-2024/) in a cell. Once you do so and press Enter or Return, the URL becomes a clickable link. It's faster than using the link tool or HYPERLINK function and is especially useful if you don't necessarily need the link to look attractive.

Related: [How to Paste Text Without Formatting Almost Anywhere](https://facebook-videos.techidaily.com/new-ultimate-collection-20-chuckle-inducing-detention-cell-photos-for-a-better-day-for-2024/) 

 There is one setting you'll need to enable for this to work, though. It's a hyperlink setting that may be enabled by default, but you should check anyway to be certain it hasn't been disabled. To check the setting, select File > Options and choose "Proofing" on the left. On the right, click "AutoCorrect Options."
